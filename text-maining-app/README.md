## AIテキストマイニング（ワードクラウド）

議事録やオンライン会議のコメントを入力すると、頻出キーワードを可視化するStreamlitアプリです。

## 使い方（ローカル・uv）

```powershell
uv sync
uv run streamlit run main.py
```

## 使い方（画面操作）

1. テキストボックスに解析したい文章を貼り付けます。
2. `Analyze` を押すと、ワードクラウドとキーワード一覧が表示されます。
3. 必要に応じて以下を調整します。
   - 言語（Auto / Japanese / English）
   - 最小文字数
   - 最大表示語数
   - 品詞（日本語のみ）
   - 背景色とカラーマップ
4. 画像（PNG）やキーワード一覧（CSV）をダウンロードできます。

## 日本語テキストの注意点

- 日本語のワードクラウド表示には日本語フォントが必要です。
- Windows/Linuxで代表的なフォントを自動検出します。
- 文字が四角く表示される場合は、サイドバーからフォントファイル（ttf/otf/ttc）をアップロードしてください。

## できていること

- テキスト入力からワードクラウドを生成
- 頻出キーワードの表形式表示（上位件数）
- 日本語/英語の自動判定と手動切替
- 日本語：Janomeでの形態素解析（名詞/形容詞/動詞の抽出）
- 英語：正規表現によるトークン化（数値含有の可否切替）
- 画像（PNG）とキーワード一覧（CSV）のダウンロード
- フォント自動検出と手動アップロード

## まだできていないこと

- 複数ファイル（CSV/議事録ファイル等）の一括解析
- ストップワードのUI編集・保存
- 組織固有の辞書（専門用語）の登録
- フレーズ（n-gram）や共起語の分析
- 結果の履歴保存、共有URL、権限管理

## Streamlit Community Cloudへの公開

1. GitHubにこのリポジトリをpushします。
2. Streamlit Community Cloudで新規アプリを作成します。
3. エントリーポイントを `main.py` に設定します。
4. デプロイします。
